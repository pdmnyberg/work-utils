# Core technologies of Bird ringing

- UI using Next JS, and bootstrap
    - A nice startingpoint for structure of Typescript and CSS
- Backend using Django and with Django Rest Framework
    - A convenient way to get a good initial structure for the Model, API and Auth

# Current state of Bird ringing

- Frontend app structure stable
    - Navigation and most view are in place
    - Some actions and subviews are still missing
- Backend app structure getting close to stable
    - Model is mostly stable
    - Django Rest Framework has proven nice to work with
    - Auth is pretty convenient to apply
